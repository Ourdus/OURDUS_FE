## ✏️ 프로젝트 사용 방법


1. cloning
```
git clone https://github.com/Ourdus/OURDUS_FE.git
```

2. install

- 2-1 yarn 최신 업데이트
   ```
   yarn install
  ```

3. Execute 

```
 yarn start
```
